Desktop App

Desktop App Overview

The desktop app is a graphical front end for the same review engine the terminal uses. It trades the terminal's instant, minimal feel for a richer surface: split diffs, an embedded terminal and browser, a multi-model review “arena”, point-and-click settings, and persistent tabs. Everything shared with the terminal is covered under Core Concepts; this section is the desktop-specific parts.

▲

The desktop app is in active development. It is built from source today (it is not part of the published releases yet), and details such as exact keyboard shortcuts may change. See Installation for build steps.

How it relates to the terminal app

Same engine, same data. The desktop app reads and writes the same review files per repository, branch, and view, so a review started in the terminal continues in the desktop app and vice versa. The mental model — diff modes, watch mode, reviewed marks, questions vs GitHub comments, AI sidecars and staleness — is identical. What differs is the presentation and a set of features only a GUI can offer.

The layout

The window is organized into several regions:

RegionPurpose
Tab strip (top)Open repos, branches, and PRs as tabs. Drag to reorder, middle-click to close, jump with Cmd+1…9.
Branch context barCurrent branch, base comparison, PR number, change summary, and a Local Branch ↔ PR Diff toggle.
Left sidebarCollapsible file tree with search, plus a project / branch / PR picker for switching context.
Diff view (center)Unified or split diff with inline threads, AI findings, and intra-line word-diff highlighting.
Right panelThree tabs — Branch (git/PR status), Review (AI findings, summary, arena history), and Notes (questions & GitHub comments).
Browser (optional)An embedded webview for docs, design systems, or APIs, with an annotation mode.
Terminal drawer (bottom)A resizable embedded shell for running commands without leaving the app.

Command surfaces

Two palettes make everything reachable without hunting through menus:

  • Command palette (Cmd+K) — a searchable list of actions, navigation, and files in the current diff.
  • AI action palette / AI Hub (Cmd+A) — run a review, launch the arena, pick a provider and model, choose specialized reviewers, export, and more.

Running it

# development shell with hot reload
./scripts/tauri-dev.sh

# with extra logging (e.g. the review arena)
./scripts/tauri-dev.sh --logs arena

# build a release bundle (.app + .dmg on macOS)
./scripts/tauri-build.sh
