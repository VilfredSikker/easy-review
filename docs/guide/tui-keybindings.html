Terminal UI

Keybinding Reference

The complete key map for the er terminal UI, grouped by task. Bottom-bar hints show a subset; this is the full list. Hint groups are configurable — see Configuration.

KeyAction

◆

Some keys are mode-dependent: r refreshes the diff normally but replies to a comment in comment focus mode, and R toggles resolved there. The status bar always reflects what is available right now.
